## Style Guidelines

**Style:** Data-Dense Dashboard — KPI cards, tables, filters, minimal padding, WCAG AA.

**Key Effects:** Hover tooltips, row highlight on hover, smooth filter animations (150–300ms), loading spinners.

**Page pattern (marketing):** Horizontal scroll journey optional; **dashboard IA** prioritizes dense civic data over hero marketing.

---

## Anti-Patterns

- Emojis as icons — use Lucide
- Missing `cursor-pointer` on clickables
- Layout-shifting hovers
- Low contrast (keep ≥ 4.5:1)
- Ornate chrome; blocking filters

---

## Pre-Delivery Checklist

- [ ] Lucide icons only; consistent set  
- [ ] Clickable elements obvious + focus visible  
- [ ] Transitions 150–300ms; `prefers-reduced-motion` respected  
- [ ] Responsive: 375, 768, 1024, 1440  
- [ ] No horizontal scroll on mobile for main content
